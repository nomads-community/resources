# Overview 
This repository contains a collection of resources to support NOMADS assay implementation, you can read more about NOMADS [here](info/NOMADS_introduction.pdf).

# Folders

- **beds:** bed files related to the various NOMADS assays ([nomadic](https://jasonahendry.github.io/nomadic/) will be the most up to date).
- **info**
    - [Equipment](info/Equipment.xlsx) required for performing NOMADS sequencing
    - [Commodities](info/Commodities.xlsx) for running the assay
    - Installing and setting up data sharing with [Insync](info/Insync_Installation.docx)
    - Setting up on [Windows](info/Windows_Installation.docx)
- **protocol:** 
    - NOMADS MVP protocol in [English](protocol/NOMADS/NOMADS_MVP_Rapid.pdf) and [French](protocol/NOMADS/NOMADS_MVP_Rapid_French.pdf)
    - NOMADS8 and NOMADS 16 protocol in [English](protocol/NOMADS/NOMADS_Pf_N8_N16_Ligation.pdf)
    - ONT protols for:
        - [Flow cell wash](protocol/ONT/flow-cell-wash-kit-exp-wsh004-WFC_9120_v1_revP_08Dec2020-minion.pdf)
        - [Ligation sequencing](protocol/ONT/ligation-sequencing-amplicons-sqk-lsk114-ACDE_9163_v114_revS_29Jun2022-minion.pdf)

        - [Rapid sequencing](protocol/ONT/Rapid_sequencing_DNA_V14-barcoding_(SQK-RBK114.24_or_SQK-RBK114.96)-minion.pdf)
